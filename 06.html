Топ 3 веб-браузеров для разработки

Firefox

Создатели одного из самых популярных браузеров предлагают версию, созданную специально для разработчиков – Firefox Developer Edition. Вот список функций, которые отличают её от версии для пользователей:


Web IDE для разработки, развертывания и отладки веб-приложений.

Функция просмотра сайта на экранах разных размеров (mobile, tablet, laptop).

Возможность определения неактивного CSS на странице.

Отличные DevTools для отладки и проверки кода.

Упрощенная система работы с сетками CSS с помощью визуализации.

Доступ к используемым на веб-странице шрифтам.

Web Audio Editor – функция для проверки узлов веб-аудио.

Мониторинг сети – просмотр всех сетевых запросов, оценка времени отклика и скорости подключения.

Также в этой версии кнопки инструментов веб-разработки вынесены прямо на панель управления. Firefox Developer Edition доступен для Windows, Linux и OS X.

Sizzy

Sizzy — это браузер с отличной функциональностью, предназначенный для ускорения рабочего процесса разработки. Создатели этого инструмента пообещали сделать его более продуктивным, и им это удалось.

Список предлагаемых Sizzy функций довольно обширен, но некоторые из них заслуживают особого внимания:

Моделирование и возможность сделать скриншот веб-страницы на нескольких устройствах одновременно.

Анализ элемента на всех устройствах с помощью функции Universal Inspect Element.

Проверка скорости загрузки веб-сайта в различных условиях.

Функция Spotlight для работы с сайтом с помощью клавиатуры.

Менеджер сессий для переключения между пользователями.

Данный браузер действительно создан для разработчиков, но доступен только на условиях ежемесячной или ежегодной платной подписки с двухнедельным бесплатным периодом.

Polypane

Polypane предлагает достаточное количество функций как для обычных пользователей так и разработчиков. При этом в данном браузере реализованы важные инструменты для создателей сайтов:


Поддержка нескольких окон просмотра для сайтов на различных устройствах, от мобильных до мониторов с разрешением 5K.

Имеет восемьдесят тестов на доступность.

Интеграция с популярными системами управления контентом (CMS).

Поддержка расширений для фреймворков React, Angular, Vue и Svelte.

Браузер доступен для бесплатного тестирования, но для использования расширенных функций в долгосрочной перспективе потребуется подписка.
